DESCRIPTION:
The lion command

USAGE:
    myapp cat [LEGS] lion <TEETH> [OPTIONS]

ARGUMENTS:
    <TEETH>   The number of teeth the lion has

OPTIONS:
                        DEFAULT
    -h, --help                  Prints help information
    -c <>                       The number of children the lion has
    -d <>            Monday, Thursday   The days the lion goes hunting
    -w, --weight []             The weight of the lion, in kgs
