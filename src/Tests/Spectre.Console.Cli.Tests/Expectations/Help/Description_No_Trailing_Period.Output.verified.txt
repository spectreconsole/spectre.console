DESCRIPTION:
The dog command.

USAGE:
    myapp dog <AGE> [LEGS] [OPTIONS]

ARGUMENTS:
    <AGE>
    [LEGS]    The number of legs.

OPTIONS:
    -h, --help            Prints help information.
    -a, --alive           Indicates whether or not the animal is alive.
    -n, --name <VALUE>
    -g, --good-boy
